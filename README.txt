GSA Admin Toolkit
=================


Introduction
------------

This project contains a collection of standalone programs that handle
some common tasks for Google Search Appliance administrators.

See the project home page for a list of programs provided, along with
a brief description:

http://code.google.com/p/gsa-admin-toolkit/


Prerequisites
-------------

The programs are written in either Java, Python on Shell. Any
required libraries that are non-standard will be documented within the
source code file for the program.


Directory structure
-------------------

All programs are separate files in the top level directory of the
project. 


Installation, setup and usage instructions
------------------------------------------

Instructions for compiling (if required) and running each program are
given at the top of the source code file for each program.
